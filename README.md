# 🏥 HealthSync - Hospital Management System

A full-stack Hospital Management System built using **Java, JSP, Servlets, and MySQL**.  
This project manages hospital operations like patients, doctors, appointments, billing, and pharmacy.

---

## 🚀 Features

- 👨‍⚕️ Doctor Management
- 🧑 Patient Registration & Records
- 📅 Appointment Booking System
- 💊 Prescription Management
- 🧾 Billing System
- 🏥 Room Management
- 🔐 Login & Authentication System
- 📊 Admin Dashboard

---

## 🛠️ Tech Stack

- Java (Core + Servlet + JSP)
- MySQL Database
- HTML, CSS, JavaScript
- Apache Tomcat Server
- JDBC

---

## 📂 Project Structure

- src/ → Java backend code (DAO, Servlets, Models)
- webapp/ → JSP pages (Frontend UI)
- WEB-INF/ → Libraries and configuration files

---

## ⚙️ How to Run

1. Import project in Eclipse / IntelliJ
2. Configure Apache Tomcat server
3. Setup MySQL database
4. Update DB credentials in `DBConnection.java`
5. Run project on server
